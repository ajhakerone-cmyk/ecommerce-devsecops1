## 📌 Project Description
This project demonstrates a complete DevSecOps pipeline using a Flask-based E-commerce web application.

The application is containerized using Docker and integrated with security tools such as:

- Bandit (SAST)
- Safety (Dependency Scan)
- Trivy (Container Security)
- Checkov (Infrastructure Security)
- OWASP ZAP (DAST)

All scans run automatically using **GitHub Actions CI/CD Pipeline**.
